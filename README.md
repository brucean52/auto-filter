## Quickstart

Install node modules in both the root and client directory

```
root directory
npm install 
cd client 
npm install
  
```

>1. `npm run dev` - Runs front and back end servers concurrently

localhost:3000 - App
localhost:4000/graphql - GraphQL Playground
